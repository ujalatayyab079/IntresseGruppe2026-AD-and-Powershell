# IntresseGruppe2026-AD-and-Powershell
Dette prosjektet inneholder en veiledning for å sette opp en virtuell maskin med Windows Server og konfigurere Active Directory. Det inkluderer opprettelse av OU-struktur, bruk av Group Policy (GPO) for bakgrunnsbilde og skjermsparer, samt oppsett av delte mapper og tilgangsstyring.
